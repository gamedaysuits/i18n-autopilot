# Cài đặt

## Yêu cầu

- **Node.js 20.11+** (native ESM)
- Một API key dịch thuật (xem bên dưới)

## Cài đặt

<Tabs>
  <TabItem value="npm" label="npm" default>

```bash
npm install --save-dev i18n-rosetta
```

  </TabItem>
  <TabItem value="npx" label="npx (no install)">

```bash
npx i18n-rosetta sync
```

  </TabItem>
  <TabItem value="pnpm" label="pnpm">

```bash
pnpm add -D i18n-rosetta
```

  </TabItem>
  <TabItem value="yarn" label="yarn">

```bash
yarn add --dev i18n-rosetta
```

  </TabItem>
</Tabs>

## Lấy API Key

Rosetta cần một backend dịch thuật. Hãy chọn một:

| Nhà cung cấp | Biến môi trường | Phù hợp nhất cho |
|----------|---------------------|----------|
| **OpenRouter** (khuyên dùng) | `OPENROUTER_API_KEY` | Các dự án nhiều nội dung, Markdown, hơn 200 mô hình |
| **Gemini** (gói miễn phí) | `GEMINI_API_KEY` | Bắt đầu với chi phí 0 đồng, chất lượng tốt |
| **OpenAI** | `OPENAI_API_KEY` | Truy cập trực tiếp GPT-4o |
| **Anthropic** | `ANTHROPIC_API_KEY` | Truy cập trực tiếp Claude, văn bản có sắc thái |
| **DeepL** | `DEEPL_API_KEY` | Các ngôn ngữ châu Âu, tích hợp sẵn mức độ trang trọng |
| **Google Translate** | `GOOGLE_TRANSLATE_API_KEY` | Khối lượng lớn các cặp key-value, hơn 130 ngôn ngữ |
| **Microsoft Translator** | `MICROSOFT_TRANSLATOR_API_KEY` | Azure, hơn 100 ngôn ngữ |
| **LibreTranslate** | `LIBRETRANSLATE_API_URL` | Tự lưu trữ (self-hosted), ưu tiên quyền riêng tư |

<Tabs>
  <TabItem value="openrouter" label="OpenRouter (recommended)" default>

[OpenRouter](https://openrouter.ai) cung cấp một API thống nhất cho hơn 200 mô hình LLM. Có sẵn gói miễn phí.

```bash
# Sign up at https://openrouter.ai, then:
export OPENROUTER_API_KEY=sk-or-v1-...
```

Hoặc thêm vào `.env.local`:

```bash
OPENROUTER_API_KEY=sk-or-v1-your-key-here
```

Phù hợp nhất cho: các dự án nhiều nội dung, dịch Markdown và các dự án cần bảo vệ nhận biết nội dung (khối mã, shortcode, biến nội suy).

  </TabItem>
  <TabItem value="google" label="Google Translate">

```bash
export GOOGLE_TRANSLATE_API_KEY=...
```

Phù hợp nhất cho: khối lượng lớn các cặp chuỗi key-value (hơn 130 ngôn ngữ). **Không khuyên dùng** cho nội dung Markdown — Google Translate không có khả năng nhận biết các khối mã, shortcode hoặc biến nội suy.

:::tip Phát hiện thông minh
Nếu chỉ thiết lập `GOOGLE_TRANSLATE_API_KEY` (không có OpenRouter key), rosetta sẽ tự động chuyển sang Google Translate. Không cần thay đổi cấu hình.
:::

  </TabItem>
</Tabs>

```bash
npx i18n-rosetta --help
```
