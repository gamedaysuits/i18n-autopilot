# การติดตั้ง

## ข้อกำหนดเบื้องต้น

- **Node.js 20.11+** (native ESM)
- คีย์ API สำหรับการแปลภาษา (ดูรายละเอียดด้านล่าง)

## ติดตั้ง

<Tabs>
  <TabItem value="npm" label="npm" default>

```bash
npm install --save-dev i18n-rosetta
```

  </TabItem>

## รับคีย์ API

Rosetta จำเป็นต้องใช้แบ็กเอนด์สำหรับการแปลภาษา โปรดเลือกหนึ่งรายการ:

| ผู้ให้บริการ | ตัวแปรสภาพแวดล้อม | เหมาะสำหรับ |
|----------|---------------------|----------|
| **OpenRouter** (แนะนำ) | `OPENROUTER_API_KEY` | โปรเจกต์ที่มีเนื้อหาเยอะ, Markdown, โมเดลมากกว่า 200+ รายการ |
| **Gemini** (ใช้งานฟรี) | `GEMINI_API_KEY` | เริ่มต้นโดยไม่มีค่าใช้จ่าย, คุณภาพดี |
| **OpenAI** | `OPENAI_API_KEY` | เข้าถึง GPT-4o โดยตรง |
| **Anthropic** | `ANTHROPIC_API_KEY` | เข้าถึง Claude โดยตรง, ข้อความที่มีความละเอียดอ่อน |
| **DeepL** | `DEEPL_API_KEY` | ภาษายุโรป, มีการตั้งค่าระดับความเป็นทางการในตัว |
| **Google Translate** | `GOOGLE_TRANSLATE_API_KEY` | คู่ key-value ปริมาณมาก, รองรับมากกว่า 130+ ภาษา |
| **Microsoft Translator** | `MICROSOFT_TRANSLATOR_API_KEY` | Azure, รองรับมากกว่า 100+ ภาษา |
| **LibreTranslate** | `LIBRETRANSLATE_API_URL` | โฮสต์ด้วยตัวเอง, เน้นความเป็นส่วนตัว |

<Tabs>
  <TabItem value="openrouter" label="OpenRouter (recommended)" default>

[OpenRouter](https://openrouter.ai) ให้บริการ API แบบรวมศูนย์สำหรับโมเดล LLM มากกว่า 200+ รายการ มีระดับการใช้งานฟรี

```bash
# Sign up at https://openrouter.ai, then:
export OPENROUTER_API_KEY=sk-or-v1-...
```

หรือเพิ่มลงใน `.env.local`:

```bash
OPENROUTER_API_KEY=sk-or-v1-your-key-here
```

เหมาะสำหรับ: โปรเจกต์ที่มีเนื้อหาเยอะ, การแปล Markdown และโปรเจกต์ที่ต้องการการปกป้องเนื้อหาแบบรับรู้บริบท (บล็อกโค้ด, ชอร์ตโค้ด, ตัวแปร interpolation)

  </TabItem>
  <TabItem value="google" label="Google Translate">

```bash
export GOOGLE_TRANSLATE_API_KEY=...
```

เหมาะสำหรับ: คู่สตริง key-value ปริมาณมาก (มากกว่า 130+ ภาษา) **ไม่แนะนำ** สำหรับเนื้อหา Markdown — Google Translate ไม่สามารถรับรู้บล็อกโค้ด, ชอร์ตโค้ด หรือตัวแปร interpolation ได้

:::tip การตรวจจับอัจฉริยะ
หากมีการตั้งค่าเพียง `GOOGLE_TRANSLATE_API_KEY` (ไม่มีคีย์ OpenRouter) rosetta จะสลับไปใช้ Google Translate โดยอัตโนมัติ ไม่จำเป็นต้องเปลี่ยนการตั้งค่าใดๆ
:::

  </TabItem>
</Tabs>
