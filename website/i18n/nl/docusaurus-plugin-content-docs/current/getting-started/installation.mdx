# Installatie

## Vereisten

- **Node.js 20.11+** (native ESM)
- Een API-sleutel voor vertalingen (zie hieronder)

## Installeren

<Tabs>
  <TabItem value="npm" label="npm" default>

```bash
npm install --save-dev i18n-rosetta
```

  </TabItem>
  <TabItem value="npx" label="npx (no install)">

```bash
npx i18n-rosetta sync
```

  </TabItem>
  <TabItem value="pnpm" label="pnpm">

```bash
pnpm add -D i18n-rosetta
```

  </TabItem>
  <TabItem value="yarn" label="yarn">

```bash
yarn add --dev i18n-rosetta
```

  </TabItem>
</Tabs>

## Een API-sleutel verkrijgen

Rosetta heeft een vertalings-backend nodig. Kies er een:

| Provider | Omgevingsvariabele | Het beste voor |
|----------|---------------------|----------|
| **OpenRouter** (aanbevolen) | `OPENROUTER_API_KEY` | Projecten met veel content, Markdown, 200+ modellen |
| **Gemini** (gratis niveau) | `GEMINI_API_KEY` | Kosteloze start, goede kwaliteit |
| **OpenAI** | `OPENAI_API_KEY` | Directe toegang tot GPT-4o |
| **Anthropic** | `ANTHROPIC_API_KEY` | Directe toegang tot Claude, genuanceerde tekst |
| **DeepL** | `DEEPL_API_KEY` | Europese talen, ingebouwde formaliteit |
| **Google Translate** | `GOOGLE_TRANSLATE_API_KEY` | Grote volumes key-value paren, 130+ talen |
| **Microsoft Translator** | `MICROSOFT_TRANSLATOR_API_KEY` | Azure, 100+ talen |
| **LibreTranslate** | `LIBRETRANSLATE_API_URL` | Zelf gehost, privacy voorop |

<Tabs>
  <TabItem value="openrouter" label="OpenRouter (recommended)" default>

[OpenRouter](https://openrouter.ai) biedt een uniforme API voor 200+ LLM-modellen. Gratis niveau beschikbaar.

```bash
# Sign up at https://openrouter.ai, then:
export OPENROUTER_API_KEY=sk-or-v1-...
```

Of voeg toe aan `.env.local`:

```bash
OPENROUTER_API_KEY=sk-or-v1-your-key-here
```

Het beste voor: projecten met veel content, Markdown-vertaling en projecten die contentbewuste afscherming nodig hebben (codeblokken, shortcodes, interpolatievariabelen).

  </TabItem>
  <TabItem value="google" label="Google Translate">

```bash
export GOOGLE_TRANSLATE_API_KEY=...
```

Het beste voor: grote volumes key-value stringparen (130+ talen). **Niet aanbevolen** voor Markdown-content — Google Translate heeft geen besef van codeblokken, shortcodes of interpolatievariabelen.

:::tip Slimme detectie
Als alleen `GOOGLE_TRANSLATE_API_KEY` is ingesteld (geen OpenRouter-sleutel), schakelt rosetta automatisch over naar Google Translate. Er is geen configuratiewijziging nodig.
:::

  </TabItem>
</Tabs>
