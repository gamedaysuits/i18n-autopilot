# Installation

## Prérequis

- **Node.js 20.11+** (ESM natif)
- Une clé d'API de traduction (voir ci-dessous)

## Installation

<Tabs>
  <TabItem value="npm" label="npm" default>

```bash
npm install --save-dev i18n-rosetta
```

  </TabItem>
  <TabItem value="npx" label="npx (no install)">

```bash
npx i18n-rosetta sync
```

  </TabItem>
  <TabItem value="pnpm" label="pnpm">

```bash
pnpm add -D i18n-rosetta
```

  </TabItem>
  <TabItem value="yarn" label="yarn">

```bash
yarn add --dev i18n-rosetta
```

  </TabItem>
</Tabs>

## Obtenir une clé d'API

Rosetta nécessite un backend de traduction. Veuillez en choisir un :

| Fournisseur | Variable d'environnement | Idéal pour |
|----------|---------------------|----------|
| **OpenRouter** (recommandé) | `OPENROUTER_API_KEY` | Projets riches en contenu, Markdown, plus de 200 modèles |
| **Gemini** (offre gratuite) | `GEMINI_API_KEY` | Démarrage sans frais, bonne qualité |
| **OpenAI** | `OPENAI_API_KEY` | Accès direct à GPT-4o |
| **Anthropic** | `ANTHROPIC_API_KEY` | Accès direct à Claude, texte nuancé |
| **DeepL** | `DEEPL_API_KEY` | Langues européennes, formalité intégrée |
| **Google Translate** | `GOOGLE_TRANSLATE_API_KEY` | Paires clé-valeur à fort volume, plus de 130 langues |
| **Microsoft Translator** | `MICROSOFT_TRANSLATOR_API_KEY` | Azure, plus de 100 langues |
| **LibreTranslate** | `LIBRETRANSLATE_API_URL` | Auto-hébergé, priorité à la confidentialité |

<Tabs>
  <TabItem value="openrouter" label="OpenRouter (recommended)" default>

[OpenRouter](https://openrouter.ai) fournit une API unifiée pour plus de 200 modèles LLM. Une offre gratuite est disponible.

```bash
# Sign up at https://openrouter.ai, then:
export OPENROUTER_API_KEY=sk-or-v1-...
```

Ou ajoutez à `.env.local` :

```bash
OPENROUTER_API_KEY=sk-or-v1-your-key-here
```

Idéal pour : les projets riches en contenu, la traduction de Markdown et les projets nécessitant une protection tenant compte du contenu (blocs de code, shortcodes, variables d'interpolation).

  </TabItem>
  <TabItem value="google" label="Google Translate">

```bash
export GOOGLE_TRANSLATE_API_KEY=...
```

Idéal pour : les paires de chaînes clé-valeur à fort volume (plus de 130 langues). **Non recommandé** pour le contenu Markdown — Google Translate n'a aucune connaissance des blocs de code, des shortcodes ou des variables d'interpolation.

:::tip Détection intelligente
Si seule `GOOGLE_TRANSLATE_API_KEY` est définie (aucune clé OpenRouter), rosetta bascule automatiquement vers Google Translate. Aucune modification de configuration n'est requise.
:::

  </TabItem>
</Tabs>

## Vérifier l'installation

```bash
npx i18n-rosetta --help
```

Vous devriez voir la liste des commandes disponibles. Vous êtes prêt pour le [Démarrage rapide](/docs/getting-started/quick-start).
