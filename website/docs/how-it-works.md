# How It Works

The full explanation of how the i18n-rosetta + MT Eval Arena ecosystem works — from competitive crowdsourced evaluation to production deployment — lives on the Arena site:

**→ [How It Works (mtevalarena.org)](https://mtevalarena.org/how-it-works)**

That document covers:

- **The problem**: why commercial MT fails low-resource and polysynthetic languages
- **The approach**: competitive crowdsourcing with reproducible evaluation
- **The ecosystem**: how the eval harness, leaderboard, and i18n-rosetta CLI connect
- **Sovereignty**: OCAP principles, cryptographic test sets, community ownership
- **How to participate**: as a researcher, developer, or language community

For i18n-rosetta CLI-specific documentation, continue with the [Quick Start](/docs/getting-started/quick-start) or explore the sidebar.
